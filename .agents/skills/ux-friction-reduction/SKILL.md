# UX Friction Reduction Skill

The system must be easy enough for daily study.

## UX principles

- Prefer 1-2 click workflows.
- Every action should be cancellable.
- AI output should be editable before insertion.
- Failed API calls should fall back to manual prompt copy.
- Avoid forcing the user to fill many fields.
- Defaults should be sensible.
- Do not interrupt reading more than necessary.

## Ask Modal ideal flow

1. Select sentence.
2. Right-click Ask.
3. Type question.
4. Click Ask AI or Copy Prompt.
5. Edit takeaway.
6. Insert Ask Card.

## Review UI ideal flow

1. Open dashboard.
2. See due card.
3. Reveal answer.
4. Rate Again/Hard/Good/Easy.
5. Next card.

## Note check ideal flow

1. Run check.
2. See score and issues.
3. Insert missing sections or generate AI check prompt.
4. Preview patches.
